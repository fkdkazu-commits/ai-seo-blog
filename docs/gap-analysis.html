ベース対応

  • 成果物（リサーチメモ・上位記事分析シート・悩み感情リスト・信頼情報源リスト）を人間が読める形で出力する

4Step

判断

人間確認・承認

未実装　最大差分

ベース対応

対応なし。全処理が完全自動のため、ブリーフ生成から執筆・公開まで人間が介入するポイントがない。

追加仕様

  • パイプラインを一時停止して人間の承認を待つ仕組みを実装する
  • 承認通知（Slack等）・承認UI・再開トリガーを実装する
  • 専門性・ブランド方針・事業目的の観点でのチェックポイントを設ける
  • 表現リスク・引用元妥当性の確認フラグを付与する
  • 「承認済みコンテンツブリーフ」「執筆時の注意事項」を出力する
追加費用 ¥3万
5Step

制作

執筆

一部対応

ベース対応

E-E-A-T準拠・5テクニック・FAQ・まとめ・関連記事を含む記事全文を一括生成する。
